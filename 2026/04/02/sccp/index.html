稀疏条件常量传播

核心是格，从def到use的传播。

有Top，Const，Bottom三类，Const其实有无穷个状态。
状态转移就是从Top 到 Const/Bottom

然后定义状态转移函数meet:

  • meet(any, Top) = any
  • meet(any, Bottom) = Bottom
  • meet(Ci, Cj) = Ci if i==j
  • meet(Ci, Cj) = Bottom if i!=j

定义evalExpr函数(不处理phi)
- 如果操作数有Bottom， 结果是Bottom
- 如果都是常数，计算
- 返回Bottom

考虑下
bb:
a = phi(0, b)
b = a + 1
jmp bb

最开始 a:T, b:T
然后a被标记为C(0), b也是C(1)
然后a = meet(C(0), C(1)) = Bottom, b: Bottom
